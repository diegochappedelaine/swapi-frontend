# SWAPI Rebels Alliance Search System

Quick project using [SWAPI](https://swapi.dev/)
This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Getting started

```bash
$> yarn
$> yarn start
```

## Stack

- React
- TypeScript
- [@chakra-ui](https://chakra-ui.com/)
